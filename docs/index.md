# Pufferfish Skills: Epic Classes Integration 

Welcome to the documentation for the Epic Classes extension for Pufferfish Skills!

This addon bridges the gap between Pufferfish Skills and the Epic Classes mod, allowing you to create rich, RPG-like classes bounded natively within Pufferfish's skill trees and datapack systems.

## Getting Started

If you are new to the mod or setting up the systems from scratch, begin with the Core Features documentation. 
*   [Core Getting Started](core_features/GETTING_STARTED.md)
*   [Universal Loot System](core_features/Universal_Loot_System.md)
*   [Skill Imbuement System](core_features/Skill_Imbuement_System.md)
*   [Toggle System](core_features/Toggle_System.md)
*   [Core Datapack Guide](core_features/DATAPACK_GUIDE.md)

## Example Environment
You can find a complete, ready-to-use template for all of these systems in the `docs/Datapack_Example/epic_class_template` folder. It includes definitions for massive skill trees, complex attributes, and complete Epic Class interfaces.
