# Contributing

Thanks for your interest in contributing to Pufferfish Skill Leveling!

## Code style

This project uses an `.editorconfig` file to enforce basic formatting. Java and Gradle files must:

- Use UTF-8 encoding
- Use LF line endings
- Use 4 spaces for indentation

Please configure your IDE to respect these settings and run auto-formatting before committing changes.

## Pull requests

1. Fork the repository and create a branch for your work.
2. Make sure `./gradlew check` passes before submitting a pull request.
3. Include clear descriptions of your changes.

Thank you for helping improve the project!
